About

 guile-dbi is a database generic interface. The main lib provide a way to use dbds (data base drivers) which are linked at run-time (like perl dbi or php or...). This provide a very flexible way to use guile scheme implementation for db works. Now, implemented main dbi interface lib, mysql and postgres plugins... In near future more drivers will be available (if someone ask for :) )

Please send comments on these web pages to guile-dbi-devel@gna.org, send other questions to guile-dbi-users@gna.org.
